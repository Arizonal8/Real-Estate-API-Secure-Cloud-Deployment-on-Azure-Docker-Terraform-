# Terraform Infrastructure Deployment

## Problems I Faced
- Terraform errors due to unsupported regions.
- Missing variables.
- Wrong resource names.
- Azure free tier limitations.

## Commands I Used
terraform init
terraform plan
terraform apply

## Key Terraform Files
### providers.tf
provider "azurerm" {
  features {}
}

### main.tf
resource "azurerm_resource_group" "rg" {
  name     = "realestate-rg"
  location = "West Europe"
}

### containerapp.tf
resource "azurerm_container_app" "api" {
  name                = "realestate-api"
  resource_group_name = azurerm_resource_group.rg.name
  container_app_environment_id = azurerm_container_app_environment.env.id
}

## Lessons Learned
Build Terraform slowly. Add resources one at a time.
